Knowit Developer Summit

24. - 25. oktober

Call for papers

Vi planlegger Knowits andre årlige felles utviklerkonferanse! Denne blir avholdt 24. - 25. oktober 2014 i Oslo, Norge eller Stockholm, Sverige.

I den forbindelse leter vi etter gode foredragsholdere til å snakke om emner som:

Enkle arkitekturer
SOA i 2013, REST, er dependency injection nødvendig?, Web services uten rammeverk, store systemer uten app servere, in-memory computing
Mobil
Mobil først design, testverktøy, multi-plattform utvikling
Devops
Infrastruktur som kode, automatisert oppsett av omgivelser (vagrant, puppet, chef), kontinuerlig deployment
Persistens
NoSQL vs. SQL (patterns, polyglot persistens), Big data, cloud, Grid
Web
*.js, responsivt design, HTML5, LESS/SASS
Testkode
Hvordan skrive gode tester, testverktøy, kontinuerlig testing, TDD, BDD
Programmering
Paradigmer, multicore-utfordringer, statisk vs dynamisk, språk (C#, Java, JavaScript, Ruby, Scala, Clojure, etc.)
Verktøy
Versjonskontroll, byggesystemer, analyseverktøy (Sonar o.l.)
... og mer!

Ditt publikum? Lidenskapelige og interesserte systemutviklere i Knowit, hovedsaklig innen Java- og .NET plattformene.

Vi ønsker oss forslag til workshops, tutorials, fulle foredrag (45 min) og lyntaler (10 min).

Vi er interessert i foredrag og andre bidrag som vil inspirere og engasjere deltagerne!

Mer informasjon om lokasjon, påmelding og andre praktiske detaljer kommer senere.
