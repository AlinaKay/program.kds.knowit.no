Om konferansen

KDS er Knowits andre årlige felles utviklerkonferanse! Denne blir avholdt 24. - 25. oktober 2014 i Stockholm, Sverige.

Tema for konferansen blir:

Arkitektur
SOA, REST, dependency injection, web services uten rammeverk, store systemer uten app servere, in-memory computing
Mobil
Mobil først design, testverktøy, multi-plattform utvikling
Devops
Infrastruktur som kode, provisioning, kontinuerlig deployment
Persistens
NoSQL, SQL, big data, cloud, grid
Web
*.js, responsivt design, HTML5, Less/Sass
Testkode
Skrive gode tester, testverktøy, kontinuerlig testing, TDD, BDD
Programmering
Paradigmer, parallelisering, typing, språk (F#, Scala, Elixi, Clojure, etc.)
Verktøy
Versjonskontroll, byggesystemer, SaaS, analyseverktøy(Sonar, R, o.l.)

... og mer!

Hvem er målgruppen? Lidenskapelige og interesserte systemutviklere i Knowit, hovedsaklig innen Java- og .NET plattformene.

Det vil bli workshops, tutorials, fulle foredrag (45 min) og lyntaler (10 min).
