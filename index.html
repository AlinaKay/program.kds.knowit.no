Kjære Knowit-ansatt

Vi planlegger Knowits første utviklerkonferanse! Denne vil bli avholdt 12-13 april 2013 i Oslo, Norge.

I den forbindelse leter vi etter gode foredragsholdere til å snakke om emner som:

Simple architectures
SOA in 2013, REST, is dependency injection necessary?, Web services without frameworks, big systems without app servers, in-memory computing
Mobile
Mobile first design, testing tools, multi-platform development
Devops
Infrastructure as code, automated setup of environments (vagrant, puppet, chef ++), continuous deployment
Persistence
NoSQL vs. SQL (patterns, polyglot persistence), Big data, cloud, Grid
Web
*.js, responsive design, HTML5, LESS/SASS/++
Test code
How to write good tests, testing tools, continuous testing
Programming
Paradigms, multicore challenges, static vs dynamic, languages (.NET, Java, JavaScript, Ruby, Scala, Clojure ++)
Tools
Version control, build tools, analysis tools (e.g. Sonar)

Ditt publikum? Lidenskapelige og interesserte systemutviklere i Knowit, hovedsaklig innen Java- og .NET plattformene.

Vi ønsker oss forslag til workshops, tutorials, fulle foredrag (45 min) og lyntaler (10 min).

Vi er interessert i lidenskapelige foredrag og andre bidrag som vil inspirere og engasjere deltagerne!
