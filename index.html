Knowit Developer Summit

24. - 25. oktober

Stockholm

Call for papers

Vi planlegger Knowits andre årlige felles utviklerkonferanse! Denne blir avholdt 24. - 25. oktober 2014 i Stockholm, Sverige.

I den forbindelse leter vi etter motiverte utviklere til å snakke om emner som:

Arkitektur
SOA, REST, dependency injection, web services uten rammeverk, store systemer uten app servere, in-memory computing
Mobil
Mobil først design, testverktøy, multi-plattform utvikling
Devops
Infrastruktur som kode, provisioning, kontinuerlig deployment
Persistens
NoSQL, SQL, big data, cloud, grid
Web
*.js, responsivt design, HTML5, Less/Sass
Testkode
Skrive gode tester, testverktøy, kontinuerlig testing, TDD, BDD
Programmering
Paradigmer, parallelisering, typing, språk (F#, Scala, Elixi, Clojure, etc.)
Verktøy
Versjonskontroll, byggesystemer, SaaS, analyseverktøy(Sonar, R, o.l.)

... og mer!

Meld forslag til foredrag, idéer eller temaer

Ditt publikum? Lidenskapelige og interesserte systemutviklere i Knowit, hovedsaklig innen Java- og .NET plattformene.

Vi ønsker oss forslag til workshops, tutorials, fulle foredrag (45 min) og lyntaler (10 min).

Vi er interessert i foredrag og andre bidrag som vil inspirere og engasjere deltagerne!

Call for presentations stenger 1.Juli 2014!
