Knowit Developer Summit

October 24 - 25

Call for papers

We are planning Knowit's second yearly company-wide software developer conference! It will be held on the 24th - 25th of October 2014 in Stockholm, Sweden.

Topics of the conference are:

Architecture
SOA, REST, dependency injection, web services without frameworks, web services without app servers, in-memory computing
Mobile
Mobile first design, testing tools, multi-platform development
Devops
Infrastructure as code, provisioning, continuous deployment
Persistence
NoSQL, SQL, big data, cloud, grid
Web
*.js, responsive design, HTML5, Less/Sass
Test code
Writing good tests, testing tools, continuous testing, TDD, BDD
Programming
Paradigms, parallelism, typing, languages(F#, Scala, Elixir, Clojure, etc.)
Tools
Version control, build systems, SaaS, analysis(Sonar, R, etc.)

... and more!

Who will be attending? Passionate and truly interested developers within the Knowit organization, mainly programming on the Java and .NET platforms for the web.

There will be workshops, tutorials, full presentations (45 min) and lightning talks (10 min).
