Knowit Developer Summit

November 13 - 14

Call for papers

We are planning Knowit's second yearly company-wide software developer conference! It will be held on the 13th - 14th of November 2015 in Stockholm, Sweden.

We are seeking motivated developers to speak at the conference about topics such as:

Architecture
Microservices, DDD, containere (Docker, Rocket...), event-sourcing, asynchronous services, fault tolerant systems, REST, dependency injection, web services without frameworks, web services without app servers, in-memory computing
Mobile
Mobile first design, testing tools, multi-platform development, Apple Watch, iOS, Android, Windows Mobile
Devops
Infrastructure as code, provisioning, continuous deployment
Persistence
NoSQL, SQL, big data, cloud, grid
Web
*.js, responsive design, HTML5, Less/Sass
Test code
Writing good tests, testing tools, continuous testing, TDD, BDD
Programming
Paradigms, parallelism, typing, languages (C#, F#, Scala, Elixir, Clojure, etc.)
Tools
Version control, build systems, SaaS, analysis (Sonar, R, etc.)
Platforms and other
Cloud, Internet of Things, CMS, CRM, Office 365, EPiServer

... and more!

Your audience? Passionate and truly interested developers within the Knowit organization, mainly programming on the Java and .NET platforms for the web.

We are open to suggestion on workshops, full presentations (60 min), shorter presentations (30 min), and lightning talks (10 min).

We are interested in talks and contributions that will captivate and inspire the participants!

More information about the venue, registration and other practicalities will be announced later.
