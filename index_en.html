Knowit Developer Summit

October 24 - 25

Call for papers

We are planning Knowit's second yearly company-wide software developer conference! It will be held on the 24th - 25th of October 2014 in Oslo, Norway or Stockholm, Sweden.

We are seeking the best and brightest to speak at the conference about topics such as:

Simple architectures
SOA in 2013, REST, is dependency injection necessary?, Web services without frameworks, big systems without app servers, in-memory computing
Mobile
Mobile first design, testing tools, multi-platform development
Devops
Infrastructure as code, automated setup of environments (vagrant, puppet, chef), continuous deployment
Persistence
NoSQL vs. SQL (patterns, polyglot persistence), Big data, cloud, Grid
Web
*.js, responsive design, HTML5, LESS/SASS
Test code
How to write good tests, testing tools, continuous testing, TDD, BDD
Programming
Paradigms, multicore challenges, static vs dynamic, languages (C#, Java, JavaScript, Ruby, Scala, Clojure, etc.)
Tools
Version control, build tools, analysis tools (e.g. Sonar)

... and more!

Your audience? Passionate and truly interested developers within the Knowit organization, mainly programming on the Java and .NET platforms.

We are open to suggestion on workshops, tutorials, full presentations (45 min) and lightning talks (10 min).

We are interested in talks and contributions that will captivate and inspire the participants!

More information about the venue, registration and other practicalities will be announced later.
